# Super Gulp Demo

Super Gulp demo.

## Installation

```sh
npm install -g gulp-cli
npm install
```

## Usage

Original commands:

```sh
gulp
gulp test
gulp watch
gulp build
```

Aliasing `gulp` with `npm` for easy commands:

```sh
npm start
npm test
npm run watch
npm run build
```
